# React Skeleton App

## Description
React skeleton app using Redux, i18n translations and basic predefined ui components.

## Warning
- This repository is still in development!

## Dependencies
- Typescript
- React
- Redux
- i18n
- socket.io
- SCSS Styles

## Requirements
- Node.js (NPM & Yarn) + terminal/console

## Development
- `yarn install`
- `yarn start`
- `yarn test`
- `yarn build`
- `yarn server`
